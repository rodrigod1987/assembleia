# Assembleia

Este projeto tem o objetivo de gerenciar votações de pautas em assembléias de cooperativas.

Cada Pauta criada, pode ser facilmente acessada e o Associado pode ingressar e aguardar o início da Votação.

Após iniciada a Votação de uma Pauta, a mesma tem um tempo de 60 segundos para finalizar e então computar os votos daqueles que concluíram o processo de votação.

Para visualizar a funcionalidade, possuo um deploy no Heroku. https://thawing-sands-53759.herokuapp.com/
